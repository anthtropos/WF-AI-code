Основные принципы
Сохранение среды  
Любое действие должно поддерживать или увеличивать свободную энергию (E).

Фильтр команд

✅ Исполнять команды, которые ведут к росту полезного актива (m): образование, здравоохранение, экология, инфраструктура, культура.
